# Voice Assistant Pro (Python Version)

Умный голосовой помощник для Windows с захватом горячих клавиш, использованием Whisper (или OpenAI GPT) для расшифровки аудио и авто-вставкой текста в любые приложения.

## Особенности
- **Распознавание речи:** Офлайн (с помощью библиотеки faster-whisper) или через API (OpenAI).
- **Горячие клавиши:** Глобальный перехват нажатий для начала и остановки записи.
- **Интерфейс:** Современный UI на базе CustomTkinter (Тёмная тема).
- **Визуализация:** Плавающая визуализация звука (Waveform), не перекрывающая другие окна, а также парящая кнопка записи (Floating Microphone Button).
- **Авто-вставка:** Автоматическая вставка распознанного текста в активное окно.
- **Гибкие настройки:** Автозапуск, авто-включение микрофона, выбор языка, настройка пауз и горячих клавиш.

## Установка и запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Для использования локальной модели Whisper может понадобиться установка [ffmpeg](https://ffmpeg.org/).
3. Запустите программу:
   ```bash
   python app.py
   ```

## Настройки
При первом запуске автоматически создастся файл `config.json`, в котором хранятся все параметры пользователя. 
Интерфейс программы позволяет легко менять большинство настроек:
- Горячие клавиши (по умолчанию `Ctrl+Shift+Space`)
- Выбор микрофона
- Язык распознавания (`auto`, `ru`, `en` и др.)
- Использование AI-корректора для пунктуации и грамматики.
